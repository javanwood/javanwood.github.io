Automatic Node Layout

October 6 2014

This prototype lays out a set of predefined nodes automatically. I needed a solution that layed out the nodes with the minimal link crossing and similar spacing across the graph. Most importantly the solution needed to be repeatable, as the graph would be used for visualisation of a particular problem.

My first solution (5 hrs) added each node as layed out with the least path distance, but the link crossing performance was very poor. The second solution (2 hrs, below) is essentially a simple physics engine where links between nodes are springs and each node has a magical repulsive force to cause them to spread out. The algorithm runs until it reaches steady state (stops moving). Both solutions are order dependant, but that's not really a problem.

Directions: Hit run to begin/reset arranging, shuffle to change the order of the nodes, changing the arrangement.
